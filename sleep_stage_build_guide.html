Introduction

System Architecture & Core Concepts

This guide walks you through building a complete knowledge-distilled sleep staging system: from raw polysomnography data, through a high-capacity teacher model, all the way to C++ inference code running on a sub-$5 microcontroller inside a consumer EEG headband.

Sleep staging (classifying 30-second epochs into Wake, N1, N2, N3, REM) is clinically performed using full-montage PSG — 22+ channels, bulky hardware, lab environment. Your goal: match 93%+ of that accuracy using a single dry-contact EEG electrode and a coin-cell battery.

⚡ Why Knowledge Distillation beats direct small-model training
A student trained from scratch on the same data reaches ~87% accuracy. The same student, guided by a teacher's soft probability distributions (which encode rich inter-class relationships), reaches 93–94%. The teacher's uncertainty is information.

⚠ Critical: Domain Gap Management
Hospital PSG uses wet gel electrodes (impedance <5kΩ). Consumer dry EEG has 40–200kΩ contact impedance, producing 3–10× more motion artifact and 50/60 Hz noise. Your preprocessing pipeline must simulate this for teacher training — otherwise the student will fail on real wearable data.

Preprocessing Pipeline

Python preprocess/eeg_pipeline.py 
import numpy as np
import mne
from scipy import signal
from dataclasses import dataclass
from typing import Tuple, List

@dataclass
class EEGConfig:
    fs: int = 100          # Target sample rate (Hz) — downsample from 256Hz PSG
    epoch_sec: int = 30     # AASM standard epoch length
    window_sec: float = 2.0  # Spectrogram window
    overlap: float = 0.5    # 50% hop
    f_low: float = 0.5      # Bandpass low (Hz)
    f_high: float = 35.0    # Above 35Hz is EMG / power line contamination
    n_freqs: int = 129      # STFT frequency bins
    n_times: int = 29       # STFT time steps per 30-s epoch


class SleepEEGPreprocessor:
    def __init__(self, cfg: EEGConfig):
        self.cfg = cfg
        self.sos = signal.butter(
            N=4, Wn=[cfg.f_low, cfg.f_high],
            btype='bandpass', fs=cfg.fs, output='sos'
        )

    def load_and_segment(self, edf_path: str,
                          channel: str = 'EEG Fpz-Cz'
                          ) -> Tuple[np.ndarray, np.ndarray]:
        """Load EDF, pick single EEG channel, segment into 30-s epochs."""
        raw = mne.io.read_raw_edf(edf_path, include=[channel], preload=True)
        raw.resample(self.cfg.fs)
        data = raw.get_data(0).squeeze()  # shape: (N_samples,)

        epoch_len = self.cfg.fs * self.cfg.epoch_sec
        n_epochs = len(data) // epoch_len
        epochs = data[:n_epochs * epoch_len].reshape(n_epochs, epoch_len)
        return epochs  # (N, 3000) @ 100Hz

    def clean(self, epochs: np.ndarray) -> np.ndarray:
        """Bandpass → notch → robust amplitude normalisation."""
        # Butterworth bandpass
        filtered = signal.sosfiltfilt(self.sos, epochs, axis=-1)
        # Notch at 50 Hz (EU) — widen to ±3 Hz for dry electrode variability
        b, a = signal.iirnotch(50, Q=15, fs=self.cfg.fs)
        filtered = signal.filtfilt(b, a, filtered, axis=-1)
        # Robust z-score (median/IQR — immune to spike outliers)
        med = np.median(filtered, axis=-1, keepdims=True)
        iqr = (np.percentile(filtered, 75, axis=-1, keepdims=True)
               - np.percentile(filtered, 25, axis=-1, keepdims=True))
        return (filtered - med) / (iqr + 1e-8)

    def to_spectrogram(self, epochs: np.ndarray) -> np.ndarray:
        """Convert epochs to log-power spectrograms: (N, 1, F, T)."""
        cfg = self.cfg
        nperseg = int(cfg.window_sec * cfg.fs)        # 200 samples
        noverlap = int(nperseg * cfg.overlap)           # 100 samples

        specs = []
        for ep in epochs:
            f, t, Zxx = signal.stft(ep, fs=cfg.fs,
                                      nperseg=nperseg, noverlap=noverlap)
            power = np.abs(Zxx) ** 2
            # Keep only 0.5–35 Hz bins
            freq_mask = (f >= cfg.f_low) & (f <= cfg.f_high)
            power = power[freq_mask]           # (F, T)
            log_power = np.log1p(power)       # log compression
            specs.append(log_power[np.newaxis]) # (1, F, T)

        return np.stack(specs)               # (N, 1, F, T)

    def augment_dry_electrode(self, raw: np.ndarray,
                               noise_std: float = 0.15) -> np.ndarray:
        """
        Simulate dry-electrode artifacts for teacher training.
        Teaches the teacher to produce robust soft labels
        even on noisy single-channel inputs.
        """
        augmented = raw.copy()
        # 1. White noise (high contact impedance)
        augmented += np.random.randn(*raw.shape) * noise_std
        # 2. Random motion artifact bursts (realistic)
        if np.random.rand() < 0.3:
            start = np.random.randint(0, raw.shape[-1] - 50)
            amplitude = np.random.uniform(2, 5)
            augmented[..., start:start+50] += (
                amplitude * np.random.randn(50)
            )
        # 3. 50 Hz power-line interference (variable amplitude)
        t = np.arange(raw.shape[-1]) / self.cfg.fs
        pl_amp = np.random.uniform(0.1, 0.5)
        augmented += pl_amp * np.sin(2 * np.pi * 50 * t)
        return augmented

Data Split Strategy

📌 Important: Subject-Level Splits
Always split by subject ID, never by epoch. Epochs from the same subject have strong correlations — epoch-level splits cause data leakage and ~8% inflated accuracy scores. Use stratified subject splits to maintain class balance across folds.
Pythonpreprocess/splits.py
from sklearn.model_selection import GroupShuffleSplit
import pandas as pd

def make_splits(df: pd.DataFrame,
                  test_size: float = 0.15,
                  val_size: float = 0.10) -> dict:
    """
    df must have columns: subject_id, spec_path, label
    Returns train/val/test index splits (subject-stratified).
    """
    gss = GroupShuffleSplit(n_splits=1, test_size=test_size, random_state=42)
    train_val_idx, test_idx = next(
        gss.split(df, groups=df['subject_id'])
    )
    df_trainval = df.iloc[train_val_idx]

    gss2 = GroupShuffleSplit(n_splits=1,
                              test_size=val_size / (1 - test_size),
                              random_state=42)
    train_idx, val_idx = next(
        gss2.split(df_trainval, groups=df_trainval['subject_id'])
    )
    return {
        'train': df_trainval.iloc[train_idx],
        'val':   df_trainval.iloc[val_idx],
        'test':  df.iloc[test_idx]
    }

Teacher Architecture (PyTorch)

Pythonmodels/teacher_crnn.py
import torch
import torch.nn as nn
import torch.nn.functional as F

class ConvBlock(nn.Module):
    """Residual conv block with spatial & channel attention."""
    def __init__(self, in_ch, out_ch, stride=1):
        super().__init__()
        self.bn1  = nn.BatchNorm2d(in_ch)
        self.conv1 = nn.Conv2d(in_ch, out_ch, 3,
                                  stride=stride, padding=1, bias=False)
        self.bn2  = nn.BatchNorm2d(out_ch)
        self.conv2 = nn.Conv2d(out_ch, out_ch, 3,
                                  padding=1, bias=False)
        # Channel attention (Squeeze-Excitation)
        self.se = nn.Sequential(
            nn.AdaptiveAvgPool2d(1),
            nn.Flatten(),
            nn.Linear(out_ch, out_ch // 8), nn.ReLU(),
            nn.Linear(out_ch // 8, out_ch), nn.Sigmoid()
        )
        self.shortcut = (nn.Conv2d(in_ch, out_ch, 1, stride=stride, bias=False)
                         if in_ch != out_ch or stride != 1 else nn.Identity())

    def forward(self, x):
        h = self.conv2(F.elu(self.bn2(
              self.conv1(F.elu(self.bn1(x))))))
        se_w = self.se(h)[..., None, None]
        return h * se_w + self.shortcut(x)


class TeacherCRNN(nn.Module):
    """
    Full teacher: ResNet-style CNN encoder + 4-layer BiLSTM classifier.
    Input:  (B, L, C, F, T)  — B=batch, L=seq_len=20, C=1 channel, F=69 freq, T=29 time
    Output: (B, L, 5)        — 5 sleep stage logits per epoch
    """
    def __init__(self, n_classes=5, lstm_hidden=512):
        super().__init__()
        # ── CNN Backbone ─────────────────────────────────────────────
        self.encoder = nn.Sequential(
            nn.Conv2d(1, 64, kernel_size=(7, 3), padding=(3, 1), bias=False),
            nn.BatchNorm2d(64), nn.ELU(),
            nn.MaxPool2d((2, 1)),                   # (B, 64, F/2, T)
            ConvBlock(64,  128, stride=2),            # (B, 128, F/4, T/2)
            ConvBlock(128, 256, stride=2),            # (B, 256, F/8, T/4)
            ConvBlock(256, 512, stride=2),            # (B, 512, F/16, T/8)
            nn.AdaptiveAvgPool2d((1, 1)),             # (B, 512, 1, 1)
        )
        self.feat_dim = 512

        # ── Sequence Temporal Encoder (BiLSTM) ───────────────────────
        self.lstm = nn.LSTM(
            input_size=self.feat_dim,
            hidden_size=lstm_hidden,
            num_layers=4,
            batch_first=True,
            bidirectional=True,
            dropout=0.3
        )
        # ── Classifier Head ──────────────────────────────────────────
        self.head = nn.Sequential(
            nn.LayerNorm(lstm_hidden * 2),
            nn.Dropout(0.4),
            nn.Linear(lstm_hidden * 2, 256), nn.ELU(),
            nn.Linear(256, n_classes)
        )

    def forward(self, x, return_features=False):
        B, L, C, F, T = x.shape
        # Encode each epoch independently via CNN
        feats = self.encoder(
            x.view(B * L, C, F, T)
        ).view(B, L, self.feat_dim)                   # (B, L, 512)
        # BiLSTM across the sequence of epochs
        seq_out, _ = self.lstm(feats)                  # (B, L, 1024)
        logits = self.head(seq_out)                     # (B, L, 5)
        if return_features:
            return logits, feats, seq_out
        return logits

Teacher Training

Pythontrain/train_teacher.py
import torch
from torch.optim.lr_scheduler import OneCycleLR
from torch.cuda.amp import GradScaler, autocast

def compute_class_weights(labels: np.ndarray) -> torch.Tensor:
    """Inverse-frequency weighting: N3 and N1 are heavily underrepresented."""
    counts = np.bincount(labels, minlength=5)
    weights = 1.0 / (counts + 1)
    return torch.tensor(weights / weights.sum(), dtype=torch.float32)

def train_teacher(model, train_dl, val_dl, epochs=60, device='cuda'):
    model = model.to(device)
    weights = compute_class_weights(train_dl.dataset.labels).to(device)
    criterion = nn.CrossEntropyLoss(weight=weights, label_smoothing=0.05)

    optimizer = torch.optim.AdamW(model.parameters(),
                                   lr=3e-4, weight_decay=1e-4)
    scheduler = OneCycleLR(optimizer, max_lr=1e-3,
                             steps_per_epoch=len(train_dl), epochs=epochs)
    scaler = GradScaler()  # Mixed precision (2× speedup on A100)

    for epoch in range(epochs):
        model.train()
        for specs, labels in train_dl:
            specs, labels = specs.to(device), labels.to(device)
            optimizer.zero_grad()
            with autocast():
                logits = model(specs)          # (B, L, 5)
                loss = criterion(
                    logits.view(-1, 5),        # (B*L, 5)
                    labels.view(-1)            # (B*L,)
                )
            scaler.scale(loss).backward()
            scaler.unscale_(optimizer)
            nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
            scaler.step(optimizer); scaler.update()
            scheduler.step()

Student Model

Microcontroller-Deployable Student Architecture

The student is a depthwise-separable CNN followed by a single-layer GRU. It operates on single-channel spectrograms and is designed to fit in <180KB of flash and <44KB of RAM — essential for battery-powered wearables.

Pythonmodels/student_crnn.py
class DepthwiseSepConv(nn.Module):
    """MobileNet-style: depthwise conv + pointwise conv. ~9× fewer MACs."""
    def __init__(self, in_ch, out_ch, stride=1):
        super().__init__()
        self.dw = nn.Conv2d(in_ch, in_ch, 3, stride=stride,
                              padding=1, groups=in_ch, bias=False)
        self.pw = nn.Conv2d(in_ch, out_ch, 1, bias=False)
        self.bn = nn.BatchNorm2d(out_ch)

    def forward(self, x):
        return F.relu6(self.bn(self.pw(self.dw(x))))


class StudentCRNN(nn.Module):
    """
    Tiny student model for MCU deployment.
    Total params: ~106K  |  Flash: ~180KB (FP32) → ~44KB (INT8)
    RAM at inference: ~38KB (single-epoch mode, GRU state: 768B)
    """
    def __init__(self, n_classes=5, gru_hidden=96):
        super().__init__()
        self.cnn = nn.Sequential(
            nn.Conv2d(1, 16, (7, 1), padding=(3, 0), bias=False), # freq filter
            nn.BatchNorm2d(16), nn.ReLU6(),
            DepthwiseSepConv(16, 32, stride=2),
            DepthwiseSepConv(32, 64),
            DepthwiseSepConv(64, 64, stride=2),
            nn.AdaptiveAvgPool2d((1, 1))  # → (B, 64, 1, 1)
        )
        # ⚠ Causal GRU: only uses past context — valid for streaming inference
        self.gru = nn.GRU(64, gru_hidden, batch_first=True)
        self.head = nn.Linear(gru_hidden, n_classes)
        self.gru_hidden = gru_hidden

    def forward(self, x, h=None):
        """
        x: (B, L, 1, F, T)  or  (B, 1, F, T) for single-epoch MCU mode
        h: GRU hidden state — persisted between 30-s epochs on device
        Returns: logits (B, L, 5), new_h
        """
        single = x.dim() == 4
        if single:
            x = x.unsqueeze(1)  # add L=1 dim
        B, L, C, F, T = x.shape
        feats = self.cnn(x.view(B*L, C, F, T)).view(B, L, 64)
        out, h = self.gru(feats, h)   # h: (1, B, 96)
        logits = self.head(out)
        return (logits.squeeze(1), h) if single else (logits, h)

    def count_params(self):
        return sum(p.numel() for p in self.parameters() if p.requires_grad)

Core Innovation

Knowledge Distillation Training Loop

KD combines three loss signals: hard labels (ground truth), soft labels from the teacher (inter-class knowledge), and feature mimicry (intermediate representation alignment). Together they transfer the teacher's learned sleep-stage manifold to the tiny student.

🌡 Temperature Scaling Intuition
At temperature T=1, the teacher outputs near-one-hot distributions — very little dark knowledge. At T=6, soft labels become "I'm 65% sure this is N2, 28% N1, 7% N3" — this uncertainty is the gold. The student learns to replicate this nuance, not just the final hard prediction.
Pythontrain/distill.py
import torch
import torch.nn.functional as F

class KnowledgeDistillationLoss(nn.Module):
    """
    Combined loss for sleep-stage KD.
    L_total = α·L_CE + β·L_KD + γ·L_feat
    """
    def __init__(self, T=6.0, alpha=0.3, beta=0.5, gamma=0.2):
        super().__init__()
        self.T = T          # Temperature — 6.0 empirically best for sleep staging
        self.alpha = alpha   # Weight for hard CE loss
        self.beta = beta     # Weight for soft KD loss
        self.gamma = gamma   # Weight for feature mimicry

    def kl_loss(self, s_logits, t_logits):
        """KL divergence between temperature-scaled distributions."""
        p_s = F.log_softmax(s_logits / self.T, dim=-1)
        p_t = F.softmax(t_logits / self.T, dim=-1)
        # T² factor restores gradient magnitude after temperature scaling
        return F.kl_div(p_s, p_t, reduction='batchmean') * (self.T ** 2)

    def feat_loss(self, s_feat, t_feat):
        """L2 feature mimicry (project student → teacher space)."""
        return F.mse_loss(s_feat, t_feat)

    def forward(self, s_logits, t_logits, labels, s_feat, t_feat):
        L_ce   = F.cross_entropy(s_logits, labels)
        L_kd   = self.kl_loss(s_logits, t_logits)
        L_feat = self.feat_loss(s_feat, t_feat)
        total  = (self.alpha * L_ce
                  + self.beta  * L_kd
                  + self.gamma * L_feat)
        return total, {'ce': L_ce, 'kd': L_kd, 'feat': L_feat}


class FeatureProjector(nn.Module):
    """Align student CNN features (64-d) → teacher space (512-d)."""
    def __init__(self, s_dim=64, t_dim=512):
        super().__init__()
        self.proj = nn.Linear(s_dim, t_dim, bias=False)

    def forward(self, x):
        return self.proj(x)  # Discarded after distillation


def distillation_loop(teacher, student, projector, train_dl,
                        epochs=80, device='cuda'):
    teacher.eval()
    for p in teacher.parameters():
        p.requires_grad = False  # Freeze teacher weights

    kd_loss = KnowledgeDistillationLoss(T=6.0)
    optimizer = torch.optim.AdamW(
        [*student.parameters(), *projector.parameters()],
        lr=5e-4, weight_decay=1e-4
    )
    scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(
        optimizer, T_max=epochs, eta_min=1e-5
    )

    for epoch in range(epochs):
        student.train(); projector.train()
        for specs, labels in train_dl:
            specs, labels = specs.to(device), labels.to(device)
            # Teacher forward (no gradient)
            with torch.no_grad():
                t_logits, t_feats, _ = teacher(specs, return_features=True)

            # Student forward
            s_logits, _ = student(specs)
            s_logits_flat = s_logits.view(-1, 5)
            t_logits_flat = t_logits.view(-1, 5)
            labels_flat = labels.view(-1)

            # Feature alignment (project student feats → teacher space)
            B, L, C, F, T = specs.shape
            s_feat = student.cnn(specs.view(B*L, C, F, T)).view(B, L, 64)
            s_feat_proj = projector(s_feat).view(-1, 512)
            t_feat = t_feats.view(-1, 512).detach()

            total, losses = kd_loss(s_logits_flat, t_logits_flat,
                                    labels_flat, s_feat_proj, t_feat)
            optimizer.zero_grad()
            total.backward()
            nn.utils.clip_grad_norm_(student.parameters(), 1.0)
            optimizer.step()
        scheduler.step()

import torch
from torch.quantization import (quantize_dynamic, get_default_qconfig,
                                    prepare, convert, QConfigMapping)
from torch.ao.quantization.quantizer.xnnpack_quantizer import (
    XNNPACKQuantizer, get_symmetric_quantization_config)

def dynamic_quantize_gru(model) -> nn.Module:
    """
    Dynamic INT8 for GRU weights (activations stay FP32).
    Best for RNN layers where activation ranges vary widely.
    Gives ~3.8× size reduction on GRU with ~0.4% accuracy cost.
    """
    q_model = quantize_dynamic(
        model,
        qconfig_spec={nn.GRU, nn.Linear},
        dtype=torch.qint8
    )
    return q_model


def static_quantize_cnn(model, calib_loader, device='cpu') -> nn.Module:
    """
    Static INT8 for CNN layers: both weights & activations quantized.
    Requires calibration data to compute activation ranges.
    """
    model.eval()
    model_fused = torch.quantization.fuse_modules(model, [
        ['cnn.0', 'cnn.1'],         # Conv + BN → fused
        ['cnn.2.dw', 'cnn.2.bn'],  # DW-Conv + BN
        ['cnn.4.dw', 'cnn.4.bn'],
    ])
    model_fused.qconfig = get_default_qconfig('fbgemm')
    prepare(model_fused, inplace=True)
    # Calibration: forward pass through 100 representative batches
    with torch.no_grad():
        for i, (specs, _) in enumerate(calib_loader):
            if i >= 100: break
            model_fused(specs)
    convert(model_fused, inplace=True)
    return model_fused


def export_to_onnx(model, save_path='student.onnx'):
    """Export to ONNX opset 17 with dynamic batch + sequence dims."""
    dummy = torch.randn(1, 1, 1, 69, 29)  # (B=1, L=1, C=1, F=69, T=29)
    dummy_h = torch.zeros(1, 1, 96)          # GRU hidden state
    torch.onnx.export(
        model, (dummy, dummy_h), save_path,
        opset_version=17,
        input_names=['spectrogram', 'gru_h'],
        output_names=['logits', 'gru_h_out'],
        dynamic_axes={
            'spectrogram': {0: 'batch', 1: 'seq_len'},
            'logits':      {0: 'batch', 1: 'seq_len'}
        }
    )
    print(f"Exported to {save_path}")

C++firmware/sleep_inference.h
// sleep_inference.h — TFLite Micro sleep stage classifier
#pragma once
#include "tensorflow/lite/micro/micro_interpreter.h"
#include "tensorflow/lite/micro/micro_mutable_op_resolver.h"
#include "student_model.cc"   // Generated C array

// Sleep stage labels (AASM)
enum class SleepStage : uint8_t { W=0, N1=1, N2=2, N3=3, REM=4 };
constexpr const char* STAGE_NAMES[] = {"Wake", "N1", "N2", "N3", "REM"};

class SleepClassifier {
public:
    static constexpr int kFreqBins  = 69;
    static constexpr int kTimeSteps = 29;
    static constexpr int kGruHidden = 96;
    static constexpr int kNumStages = 5;
    static constexpr int kTensorArenaKB = 38;

    bool Init() {
        // Register only the ops used in this model (saves ~20KB flash)
        resolver_.Add<tflite::ops::micro::Conv2D>();
        resolver_.Add<tflite::ops::micro::DepthwiseConv2D>();
        resolver_.Add<tflite::ops::micro::AveragePool2D>();
        resolver_.Add<tflite::ops::micro::FullyConnected>();
        resolver_.Add<tflite::ops::micro::GRU>();
        resolver_.Add<tflite::ops::micro::Softmax>();
        resolver_.Add<tflite::ops::micro::Quantize>();

        model_ = tflite::GetModel(student_model_data);
        interpreter_ = std::make_unique<tflite::MicroInterpreter>(
            model_, resolver_,
            tensor_arena_, sizeof(tensor_arena_)
        );
        if (interpreter_->AllocateTensors() != kTfLiteOk) {
            return false;
        }
        // Reset GRU hidden state on boot
        std::fill(gru_state_, gru_state_ + kGruHidden, 0.0f);
        return true;
    }

    /**
     * Classify a 30-second EEG epoch.
     * @param spec   Flattened log-spectrogram: float[69*29]
     *               Values in normalised range [-4, 4]
     * @return SleepStage prediction
     */
    SleepStage Classify(const float* spec) {
        // Copy input spectrogram (scaled to INT8 range)
        int8_t* input = interpreter_->input(0)->data.int8;
        const float scale = interpreter_->input(0)->params.scale;
        const int   zp    = interpreter_->input(0)->params.zero_point;
        for (int i = 0; i < kFreqBins * kTimeSteps; ++i) {
            input[i] = static_cast<int8_t>(
                std::round(spec[i] / scale) + zp
            );
        }
        // Copy GRU state into second input tensor
        float* h_in = interpreter_->input(1)->data.f;
        std::copy(gru_state_, gru_state_ + kGruHidden, h_in);

        interpreter_->Invoke();

        // Read output logits
        const int8_t* logits = interpreter_->output(0)->data.int8;
        int predicted = std::max_element(logits, logits + kNumStages) - logits;
        // Persist GRU hidden state for next epoch
        const float* h_out = interpreter_->output(1)->data.f;
        std::copy(h_out, h_out + kGruHidden, gru_state_);
        return static_cast<SleepStage>(predicted);
    }

    void ResetSession() {
        std::fill(gru_state_, gru_state_ + kGruHidden, 0.0f);
    }

private:
    uint8_t tensor_arena_[kTensorArenaKB * 1024];
    float gru_state_[kGruHidden] = {};
    tflite::MicroMutableOpResolver<8> resolver_;
    const tflite::Model*            model_       = nullptr;
    std::unique_ptr<tflite::MicroInterpreter> interpreter_;
};

Real-Time Firmware

nRF52840 EEG Acquisition & 30-Second Epoch Processing

The firmware runs a two-core pipeline: Core 0 handles BLE + application logic, Core 1 (or the main ISR) handles ADC sampling from the ADS1299 EEG front-end at 250 SPS, filters in real-time, and triggers classification at 30-second boundaries.

C++firmware/main.cpp
#include "sleep_inference.h"
#include "eeg_frontend.h"    // ADS1299 SPI driver
#include "spectrogram.h"    // Fixed-point STFT (ARM CMSIS-DSP)
#include "ble_service.h"    // BLE GATT sleep stage characteristic
#include "nrfx_rtc.h"

constexpr int kSampleRate   = 250;  // Hz (downsampled to 100 Hz in SW)
constexpr int kEpochSamples = kSampleRate * 30;  // 7,500 raw samples/epoch
constexpr int kDsRatio      = 2;    // Downsample factor (250 → 125 → FIR → 100)

// Double buffer: fill one while classifying the other (zero dead time)
volatile int16_t raw_buf[2][kEpochSamples];
volatile int     buf_ready  = -1;
volatile uint32_t sample_n  =  0;
volatile int     write_buf  =  0;

SleepClassifier classifier;
SpectrogramEngine stft;  // ARM CMSIS-DSP FFT, 128-point, Q15 fixed-point

// Called from ADS1299 DRDY interrupt — exactly every 4 ms
extern "C" void ADS1299_IRQHandler() {
    int16_t sample = eeg_frontend::ReadSample(channel=0);
    raw_buf[write_buf][sample_n++] = sample;

    if (sample_n >= kEpochSamples) {
        buf_ready = write_buf;
        write_buf ^= 1;  // Swap buffers
        sample_n  =  0;
    }
}

int main() {
    nrf_power_mgmt_init();
    ble_service::Init();
    eeg_frontend::Init(kSampleRate);

    if (!classifier.Init()) {
        /* Blink red LED — TFLM arena too small */
        while(true) {};
    }

    float spectrogram[69 * 29];

    while (true) {
        if (buf_ready >= 0) {
            // Pre-process: bandpass + notch + downsample (FIR on CMSIS-DSP)
            const float* clean = stft.FilterAndDownsample(
                raw_buf[buf_ready], kEpochSamples
            );
            // Compute log-power STFT spectrogram in Q15 fixed-point
            stft.Compute(clean, spectrogram);  // fills 69×29 floats

            // Run sleep stage classifier (~340 ms on nRF52840 @ 64 MHz)
            SleepStage stage = classifier.Classify(spectrogram);

            // Broadcast over BLE (GATT notify)
            ble_service::NotifyStage(static_cast<uint8_t>(stage));

            buf_ready = -1;  // Signal buffer consumed
        }
        // Enter sleep mode — wake on next ADS1299 interrupt
        nrf_power_mgmt_run();
    }
}
⚡ Power Optimization: WFI Sleep Between Epochs
The classifier takes ~340 ms. The remaining 29.66 s of each 30-second epoch, the CPU executes WFI (Wait For Interrupt), drawing only 12 µA. This is the key to achieving <5 mW average power — inference duty cycle is under 1.2%.

from sklearn.metrics import (cohen_kappa_score, confusion_matrix,
                               classification_report)
import seaborn as sns
import matplotlib.pyplot as plt

def full_evaluation(model, test_dl, device='cpu'):
    model.eval()
    all_preds, all_labels, all_probs = [], [], []

    with torch.no_grad():
        for specs, labels in test_dl:
            logits, _ = model(specs.to(device))
            probs = torch.softmax(logits.view(-1, 5), dim=-1)
            preds = probs.argmax(-1)
            all_preds.extend(preds.cpu().tolist())
            all_labels.extend(labels.view(-1).tolist())
            all_probs.extend(probs.cpu().tolist())

    acc   = np.mean(np.array(all_preds) == np.array(all_labels))
    kappa = cohen_kappa_score(all_labels, all_preds)
    cm    = confusion_matrix(all_labels, all_preds, normalize='true')

    print(f"Accuracy : {acc*100:.2f}%")
    print(f"Kappa    : {kappa:.4f}  (≥0.80 = substantial agreement)")
    print(classification_report(all_labels, all_preds,
          target_names=['Wake', 'N1', 'N2', 'N3', 'REM']))

    # Confusion matrix heatmap
    fig, ax = plt.subplots(figsize=(7, 6))
    sns.heatmap(cm, annot=True, fmt='.2f', cmap='Blues',
                xticklabels=['W', 'N1', 'N2', 'N3', 'REM'],
                yticklabels=['W', 'N1', 'N2', 'N3', 'REM'])
    plt.title('Student INT8 — Normalised Confusion Matrix')
    plt.savefig('confusion_matrix.pdf', dpi=300, bbox_inches='tight')

    return {'accuracy': acc, 'kappa': kappa, 'confusion': cm}

Hardware Testing Checklist

  • 1

    Tensor Arena Size Verification

    Run interpreter.arena_used_bytes() after AllocateTensors(). Must be ≤ kTensorArenaKB × 1024. If it fails, reduce GRU hidden dim by 16 (impacts accuracy ~0.3%).

  • 2

    GRU State Persistence Test

    Feed two consecutive identical spectrograms. Output probabilities must differ (second call uses the updated hidden state). A bug where state is reset every epoch drops accuracy by ~2–4%.

  • 3

    Fixed-Point STFT Numerical Validation

    Compare CMSIS-DSP Q15 spectrogram output against Python scipy.signal.stft on the same 30-second buffer. Max absolute error should be <2% after dequantization.

  • 4

    Power Profiling with PPK2

    Use the Nordic Power Profiler Kit II. Verify average current ≤3.5 mA from 3.3V supply (~11.5 mW total system including ADC). Inference spikes should be visible as ~340 ms pulses every 30 seconds.

  • 5

    Overnight Human Pilot Study

    Recruit 5 subjects for simultaneous PSG + wearable recording. Compute epoch-by-epoch agreement. Target: ≥85% epoch agreement (human-expert inter-rater agreement is 82–85%).
